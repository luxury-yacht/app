# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

In order to keep our community of users safe, please **do not** open an issue or otherwise report the vulnerability publicly. Instead, email [security@luxury-yacht.app](mailto:security@luxury-yacht.app) with a detailed description of the vulnerability.

You will be notified via email whether the vulnerability has been accepted or declined. If accepted:

- We'll do our best to fix the vulnerability in a timely manner.
- The vulnerability will be publicly disclosed in the release notes of the release that fixes the vulnerability.
- We will credit you (with your consent) in the release notes for identifying the vulnerability.

## Bug Bounties

Luxury Yacht is a free application. While we greatly appreciate your time and effort, this is not a commercial app, and there is no budget to pay bug bounties.
